### ⏱ 개발 기간
> - 22.11.18 - 22.11.25

### 📋 야구 규칙
> - Strike 3개 : Out 1개
> - Ball 4개 : 출루 1개
> - 출루 4개 : 1점
> - Out 3개 : 공수교대
> - Homerun : 출루 인원 + 타자 = 점수<br/> ex) 출루 2명일 때, 타자가 홈런을 친다면 => 3점
